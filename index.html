Class Calendar Script

This script plots all your regular day and 8 period day class rotations into a Google Calendar. This updated script is the work of Anvay Mathur. Buy him a coffee.

*Note - changes to the base calendar, daily rotation and irregular days may cause inaccurate plots.

Class information

Enter short class titles exactly as you want them to appear on a Google Calendar into the boxes below that correspond with their period. Leave the box empty if you have no class scheduled that block.

For the location box, enter the corresponding room number (e.g. H321). You can also leave the location blank if you don't need it.

For the "Set Event as Busy" checkbox, you can check the box to make the corresponding class event set as busy in your calendar.
